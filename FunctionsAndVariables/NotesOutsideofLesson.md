## Numbers
    Python supports 2 types of numbers intergers (ex: 1) and floats (Ex: 0.3)

defining a Integer in Python: 

        ```
         int = 7
         print(int)
        ```

definind a float point number

        ```
        float = 7.0
        print(float)
        ```

        ```
        myFloat = float(7)
        print(myFloat)
        ```
